# Beyond Diagonal Reconfigurable Intelligent Surfaces Utilizing Graph Theory: Modeling, Architecture Design, and Optimization

This repository contains code to reproduce the paper:

> M. Nerini, S. Shen, H. Li, B. Clerckx, "[Beyond diagonal reconfigurable intelligent surfaces utilizing graph theory: Modeling, architecture design, and optimization](https://ieeexplore.ieee.org/document/10453384)," IEEE Trans. Wireless Commun., 2024.

## Code

The script `main_Fign.m` reproduces Fig. *n*, for *n* = 6, 9, 10.

The script `main_Fig7_8.m` reproduces Fig. 7 with `NT = 2` and Fig. 8 with `NT = 8`.
